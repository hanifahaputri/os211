TOP 10 LINKS WEEK 02
1. [Trojan Virus](https://us.norton.com/internetsecurity-malware-what-is-a-trojan.html) <br>
Trojan adalah malicious code (kode penjahat) yang merusak sistem komputer. Istilah "Trojan" sendiri berasal dari sejarah Yunani dimana prajurit Yunani bersembunyi dalam kuda kayu raksasa, dan memasuki kota Troya untuk memenangkan perang. Seperti sejarahnya, virus Trojan seringkali menipu user untuk menjalankan programnya. Akan tetapi, Trojan tidak dapat mereplikasi dirinya sendiri.
2. [DDoS Attack](https://www.goldenfast.net/blog/ddos-adalah/)<br>
Distributed Denial of Service (DDoS) salah satu jenis cyber-attacks yang menyasar website, layanan online, maupun jaringan dengan cara membanjirinya dengan fake traffic yang sangat banyak. Tujuan dari serangan DDoS adalah agar server down dan tidak bisa beroperasi. Ada beberapa indikasi bahwa sebuah website terkena DDoS: Website tiba-tiba menjadi lambat, ada traffic yang tidak wajar, serta muncul spam email yang banyak.
3. [Autentikasi Biometrik](https://www.thalesgroup.com/en/markets/digital-identity-and-security/government/inspired/biometrics)<br>
Biometrik merupakan autentikasi yang berdasarkan data yang dapat diperiksa, dilakukan dengan cara membandingkan karakteristik seorang user dengan template biometrik mereka untuk menentukan apakah sesuai atau tidak. Ada dua tipe autentikasi biometrik: Morfologikal, seperti sidik jari, bentuk tangan, bentuk muka, mata, dan Biologis, seperti DNA, darah, air liur, atau urin (biasa digunakan oleh forensik).
4. [7 Things to Know About Cryptocurrency](https://www.nerdwallet.com/article/investing/cryptocurrency-7-things-to-know)<br>
Cryptocurrency adalah salah satu metode pembayaran online. Konversi dari uang asli ke cryptocurrency dibutuhkan agar kita mendapatkan melakukan transaksi. Cryptocurrency menggunakan teknologi blockchain yaitu dimana komputer mengelola dan merekam transaksi, salah satu daya tarik dari cryptocurrency adalah keamanannya. Beberapa trading cryptocurrency adalah: Bitcoin, Etherium, dan Tether. 
5. [Beginner's Guide to C](https://www.freecodecamp.org/news/the-c-beginners-handbook/)<br>
C adalah salah satu bahasa pemrograman yang paling dikenal. C dikembangkan pada tahun 1972-1973 oleh Dennis Ritchie yang dijalankan di Unix. C adalah compiled programming language; C menghasilkan file binary yang bisa dieksekusi dan didistribusi. C adalah bahasa pemrograman yang imperative procedural, dimana C dirancang untuk mendapatkan low-level akses ke memory dengan runtime minimal.
6. [Belajar Menggunakan GNUPG](https://medium.com/kode-dan-kodean/belajar-memakai-gnu-privacy-guard-gnupg-gpg-3944e19dba91)<br>
GNUPG adalah pengganti perangkat lunak gratis untuk rangkaian perangkat lunak kriptografi PGP Symantec yang dikembangkan oleh Werner Koch dan dirilis pada tahun 1999. GNUPG adalah enkripsi hybrid karena menggunakan symmetric-key cryptography untuk speed dan public-key cryptography untuk kemudahan pergantian secure key. GNUPG mengenkripsi key pair yang di-generate oleh pengguna. 
7. [Sandbox Security Defined](https://www.forcepoint.com/cyber-edu/sandbox-security#:~:text=In%20cybersecurity%2C%20a%20sandbox%20is,the%20host%20device%20or%20network.)<br>
Sandbox Security adalah strategi managemen perangkat lunak yang mengisolasi aplikasi dari sistem resource dan program lain, sandboxing memberikan layer yang dapat menyelamatkan aplikasi tersebut dari malware atau apapun yang dapat memberikan dampak buruk pada aplikasi. Sandbox memiliki tiga variasi implementasi: Emulasi Sistem, Emulasi Sistem Operasi, serta Virtualisasi.
8. [What is Code Signing](https://docs.microsoft.com/en-us/previous-versions/windows/internet-explorer/ie-developer/platform-apis/ms537361(v=vs.85)?redirectedfrom=MSDN)<br>
Code Signing adalah proses menandatangani kode executable secara digital untuk mengonfirmasi pencipta software dan memastikan bahwa program tidak dimanipulasi atau diubah. Code Signing menggunakan cryptographic hash untuk memvalidasi integritas. Biasanya digunakan untuk menyediakan keamanan saat men-deploy program. Checksum digunakan untuk memverifikasi bahwa program belum dimanipulasi.
9. [Trapdoors](https://www.techwalla.com/articles/what-is-a-computer-trapdoor)<br>
Trapdoor, seperti namanya, menyediakan pintu rahasia yang memperbolehkan seseorang untuk mengakses program jika mereka mengetahui pintu tersebut tanpa harus melewati prosedur keamanan yang dilakukan kepada pengguna lainnya. Pada umumnya, Trapdoor digunakan oleh developer untuk melakukan testing atau debugging, namun kegunaan dari Trapdoor seringkali disalahgunakan oleh programmer untuk mendapatkan akses yang seharusnya tidak mereka miliki.
10. [Protection Rings](https://www.geeksforgeeks.org/protection-ring/)<br>
Protection Rings adalah hierarki dari perlindungan domain, merupakan sebuah mekanisme untuk melindungi data dan meng-improve fungsionalitas. Hierarki dari Protection Rings diurutkan dari yang paling rendah (least privileged) ke yang paling tinggi (most privileged). Terdapat empat level, Level 3 paling rendah dan Level 0 paling tinggi. Biasanya, Level 3 mencakup aplikasi, Level 2 dan Level 1 mencakup device driver, dan Level 0 mencakup Kernel.
